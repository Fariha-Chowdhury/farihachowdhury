Featured Works

Patent: A Method for Purifying Urban Air Using a Network of Intelligent Street Lamps

This project proposes a network of solar-powered smart street lamps using photocatalytic oxidation (PCO) and real-time sensors to reduce pollutants like NOx, VOCs, and particulates. Beyond technical innovation, the project focused on human-centered urban design, which demanded attention to usability, public engagement, real-time feedback, and data visualization.
